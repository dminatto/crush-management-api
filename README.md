Api create to learn Type Script based in the [article](https://medium.com/nerdzao/crie-uma-api-restful-em-nodejs-para-gerenciar-seus-crushs-c4c74c3db96e) using MongoDB to manage fictional crushes.

## Getting started

- [NodeJS](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [MongoDB](https://www.mongodb.com/)

## Developing

``````
git clone https://github.com/dminatto/crush-management-api.git
``````
cd crush-management-api
```
npm i
```

## Building

````
npm start
```
